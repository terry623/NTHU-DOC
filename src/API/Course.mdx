# Course

課程資料
`GET /course/:id`

> 基本與校方提供 JSON 類似

> http://curricul.web.nthu.edu.tw/files/13-1073-111356.php?Lang=zh-tw

> 但移除 tab, newline, slash 等符號

```javascript
{
  data: [
    {
      科號,
      課程中文名稱,
      課程英文名稱,
      學分數,
      人限,
      新生保留人數,
      通識對象,
      通識類別,
      授課語言,
      備註,
      停開註記,
      教室與上課時間: [
        { 教室, 上課時間 },
        { 教室, 上課時間 },
        { 教室, 上課時間 },
      ],
      授課教師: [
        { 中文名稱, 英文名稱 },
        { 中文名稱, 英文名稱 },
        { 中文名稱, 英文名稱 },
      ],
      擋修說明,
      課程限制說明,
      第一二專長對應: [], // 把 tab 分隔改成 array
      學分學程對應: [], // 把 slash 分隔改成 array
      不可加簽說明,
      必選修說明: [], // 把 tab 分隔改成 array
      統計: { 目前選上人數, 目前待亂數人數 }, // new
      開課代碼, // new
    },
  ],
}
```
